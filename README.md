# Desire Look — Bridal Makeup Studio & Beauty Salon

A fast, static, SEO-optimized website for **Desire Look**, a bridal makeup studio & beauty salon in Dhanbad, Jharkhand.

## Features
- Image-led hero banner with photo collage
- Services, individual service pages, gallery, blog
- Flickity carousels (services, reviews) with arrows
- Booking popup (lead form) → WhatsApp with UTM lead-source tracking
- Google Maps embed + Google Business Profile link
- Fixed header, mobile off-canvas menu, floating Call / WhatsApp / Book buttons
- LocalBusiness schema, sitemap.xml, robots.txt

## Tech
Plain static HTML/CSS/JS. Service, blog, gallery, sitemap and robots files are generated by `generate.js`:

```bash
node generate.js
```

`index.html` and `services.html` are hand-authored; everything else is generated.

## Local preview
Serve the folder with any static server, e.g.:

```bash
npx serve .
# or
python -m http.server 8080
```

Then open http://localhost:8080
